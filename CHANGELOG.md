# 更新日志

## [1.1.0] - 2025-01-11

### ✨ 新增功能

#### 翻页动画系统
- **4种翻页动画效果**
  - 滑动动画（Slide）：流畅的左右滑动切换
  - 渐隐动画（Fade）：优雅的淡入淡出效果
  - 翻书动画（Curl）：真实的3D翻页效果
  - 无动画（None）：快速直接切换

- **动画设置**
  - 可自定义动画速度（0.1秒 - 0.8秒）
  - 支持实时切换动画类型
  - 动画说明和使用建议

- **用户体验优化**
  - 翻页时震动反馈
  - 流畅的动画过渡
  - 智能方向判断

### 🔧 技术改进

- 新增 `PageTransitionManager.swift` 翻页动画管理器
- 新增 `ChapterContentView` 带动画的章节内容视图
- 实现多种 ViewModifier 动画修饰器
- 优化阅读界面的动画性能
- 改进设置界面，添加动画选项

### 📝 文档更新

- 新增 `PAGE_ANIMATION_GUIDE.md` - 翻页动画详细指南
- 新增 `ANIMATION_DEMO.md` - 动画效果演示
- 更新 `README.md` - 添加翻页动画说明
- 更新项目结构文档

### 🎨 设计优化

- 动画曲线采用 easeInOut 缓动函数
- 动画与内容切换完美同步
- 支持不同性能设备的动画降级

## [1.0.0] - 初始版本

### ✨ 核心功能

- **竖排阅读**
  - 从上往下、从右往左的传统阅读方式
  - 完整还原古人阅读体验

- **宣纸质感界面**
  - 精美的宣纸纹理背景
  - 温暖的米黄色调
  - 墨色和朱砂红点缀

- **书库浏览**
  - 5大分类（经典、历史、哲学、文学、诗词）
  - 10部经典古籍示例
  - 书籍详情和预览

- **下载功能**
  - 一键下载书籍
  - 离线阅读支持
  - 本地数据管理

- **阅读设置**
  - 字体大小调节（14-28）
  - 4种阅读主题（宣纸、白纸、复古、夜间）
  - 章节导航和目录

- **用户系统**
  - 简单的登录/注册
  - 个人中心
  - 我的书架管理

### 📚 内容

内置10部经典古籍：
1. 论语（经典）
2. 道德经（经典）
3. 孟子（经典）
4. 史记（历史）
5. 资治通鉴（历史）
6. 唐诗三百首（诗词）
7. 宋词三百首（诗词）
8. 古文观止（文学）
9. 庄子（哲学）
10. 易经（哲学）

### 🛠 技术特点

- SwiftUI 现代化界面
- MVVM 架构设计
- Canvas 绘图技术
- 本地数据持久化
- 响应式状态管理
