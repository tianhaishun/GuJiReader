## 🔖 设计理念总结

### 宋代美学的四个核心

#### 1. 雅致
```
❌ 不张扬、不浮夸
✅ 含蓄内敛、温和自然
```

#### 2. 留白
```
❌ 不拥挤、不繁复
✅ 大量空间、呼吸感
```

#### 3. 质朴
```
❌ 不华丽、不矫饰
✅ 材质真实、自然
```

#### 4. 理性
```
❌ 不混乱、不随意
✅ 结构清晰、逻辑
```
